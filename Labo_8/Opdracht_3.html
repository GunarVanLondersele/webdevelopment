de text wordt door div gecentreerd in het css stylesheet vertical align geeft een fout als we kijken in de dev tools: invalid property value vertical align: middle gaat het volledige object van de div gaan centreren line height: als men deze gelijksteld aan de ingestelde hoogte van het object dan wordt dit mooi gecentreerd. Let op, dit moet per class appart ingesteld worden. in dit geval is dit area(100px), area short(50px), area tall(150px). om alles snel te gaan uitlijnen kan je display: table-cell gaan gebruiken in combinatie met verticale alignment: middle area tall krijgt voorang op area omdat deze na area staat gecodeerd. als we de volgorde omwisselen in het stylesheet dan wordt er geen rekening meer gehouden met tall.
